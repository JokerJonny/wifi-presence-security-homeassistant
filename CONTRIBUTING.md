# Contributing

Thank you for your interest in contributing to the NeoSHADE Home Security project.

This project aims to build powerful **privacy-first smart home security tools** using open technologies.

## Ways to Contribute

You can help by contributing:

• code improvements  
• bug fixes  
• documentation updates  
• new integrations  
• ESP32 / IoT sensor support  
• Home Assistant automations
