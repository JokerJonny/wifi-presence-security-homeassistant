# Code of Conduct

## Our Commitment

We are committed to creating an open, respectful, and collaborative environment for everyone contributing to the NeoSHADE project.

Participants are expected to act professionally and respectfully.

## Expected Behavior

• be respectful and constructive  
• welcome new contributors  
• focus on improving the project  
• communicate openly and honestly

## Unacceptable Behavior

The following behaviors will not be tolerated:

• harassment or discrimination  
• personal attacks  
• abusive language  
• intentionally harmful contributions

## Enforcement

Project maintainers may remove or reject contributions that violate this code of conduct.

## Reporting Issues

If you experience or witness unacceptable behavior, please contact:

security@neo-shade.com
