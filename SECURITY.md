# Security Policy

## Supported Versions

This project is under active development.  
Security updates will be applied to the latest version of the repository.

| Version | Supported |
|--------|-----------|
| Latest | Yes |
| Older  | No |

## Reporting a Vulnerability

If you discover a security vulnerability related to this project, please report it responsibly.

Contact:

security@neo-shade.com

or open a **private security advisory** on GitHub.

Please include:

• description of the vulnerability  
• reproduction steps  
• potential impact  
• suggested fix (if available)

Responsible disclosure helps keep the community safe.

## Security Philosophy

NeoSHADE systems are designed around:

• local-first architecture  
• minimal cloud dependency  
• transparent open-source code  
• user-controlled data
